# nest code generator

hygen.io base nest.js code generator


## usage
```shell script
yarn generate {generate-type} {name}
```


```shell script
yarn generate input-type user/create-one
```

this will generate code at ```src/user``` directory


## generate types 
* input-type
* object-type
* interface-type
* mutation-decorator
* query-decorator
* enum-type
